# UI/UX Specifications & Assets

This directory contains resources related to application layout, high-fidelity mockups, asset libraries, and the central design system.

## Design System Tokens
- **Primary Color (Dark Slate)**: `#0F172A`
- **Secondary Color (Active Blue)**: `#3B82F6`
- **Background Color (Slate-50)**: `#F8FAFC`
- **Text Color (Slate-900)**: `#0F172A`
- **Typography**: Inter (Default), Outfit (Headers).

## Component Guidelines
- Use unified spacing tokens (8px grid system).
- Implement interactive elements with hover and click micro-animations to enhance user experience.
